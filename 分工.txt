泛型

泛型（Generics）是指在定义函数、接口或类的时候，不预先指定具体的类型，而在使用的时候再指定类型的一种特性。

使用泛型来创建可重用的组件，一个组件可以支持多种类型的数据。

泛型函数

function identity<T>(arg: T): T {
    return arg;
}
identity<string>('params1')
identity<number>(100）
